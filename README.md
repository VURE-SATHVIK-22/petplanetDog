# Petplanet Dog Clinic - Premium Motion Design

A sophisticated, high-end veterinary website for Dr. T Suresh Babu, featuring premium motion design with pinned sections, smooth scroll, and cinematic aesthetics. Built with React, GSAP, Lenis smooth scroll, and Framer Motion.

## 🎬 Animation System Overview

This project implements a high-end interaction pattern featuring:

- **Lenis Smooth Scroll** - Inertial, butter-smooth scrolling
- **Pinned Section Behavior** - Images pin while content scrolls independently  
- **Staggered Reveals** - Premium timing for content entrance
- **Glassmorphism** - Modern, translucent UI elements
- **GSAP ScrollTrigger** - Advanced scroll-linked interactions

## 🌊 Core Features

### 1. Pinned Section System
Sections on the Surgeries page pin clinical categories while the detailed content scrolls independently, providing a deep-dive experience without losing context.

### 2. Clinical Excellence Gallery
A specialized gallery for orthopedic surgeries, diagnostics, and team expertise with an interactive lightbox for detailed viewing.

### 3. Holistic SEO Architecture
Comprehensive metadata, Open Graph tags, and Local Business Schema optimized for Hyderabad veterinary services.

## 🛠 Tech Stack

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **Animation**: GSAP (GreenSock), Framer Motion, Lenis
- **Icons**: Lucide React
- **Routing**: React Router v7

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server  
npm start

# Build for production
npm run build
```
