# Dr. T Suresh Babu Veterinary Clinic
### *Premium Canine Diagnostics, Surgery & Compassionate Care*

![Cinematic Banner](https://images.unsplash.com/photo-1583337130417-3346a1be7dee?q=80&w=1200&auto=format&fit=crop)

A high-performance, cinematic web experience for **Dr. T Suresh Babu** (formerly Pet Planet Dog Clinic) in Hyderabad. This project showcases advanced veterinary diagnostics and surgical excellence through a premium, motion-driven interface.

---

## 🏗️ Technical Architecture

This site is built with a **Motion-First** philosophy, prioritizing fluid performance and premium aesthetics:

- **Frontend**: [React 18](https://react.dev/) for component-driven reactivity.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for a sleek, glassmorphic design system.
- **Animation**: [GSAP (GreenSock)](https://greensock.com/) & [Framer Motion](https://www.framer.com/motion/) for orchestrated, scroll-triggered reveals.
- **Smooth Scroll**: [Lenis](https://lenis.darkroom.engineering/) for inertial, parallax-ready scrolling.
- **Icons**: [Lucide React](https://lucide.dev/) for crisp, scalable clinical iconography.

---

## ✨ Elite Features

### 🎞️ Cinematic UI/UX
- **Dual-Ring Custom Cursor**: Interactive cursor with `mix-blend-difference` for high-end feel.
- **Atmospheric Grain & Vignette**: Applied global film grain and radial vignette for cinematic depth.
- **Magnetic Hero Interactions**: Mouse-following 3D magnetic effect on key clinical imagery.

### 🏥 Clinical Specializations
- **Surgical Excellence**: Dedicated sections for orthopedic and soft tissue surgeries.
- **Advanced Diagnostics**: Showcasing digital microscopy, X-ray, and ultrasound capabilities.
- **Critical Care**: Professional highlighting of emergency and inpatient services.

### 📍 Multi-Branch Management
Comprehensive location data and dedicated pages for all clinics:
1. **Vanasthalipuram (Main)** - Hospital & Surgical Center.
2. **SR Nagar & Hasthinapuram** - Diagnostic & Wellness.
3. **Turkayamjal** - Pet Planet Dog Clinic Branch.

---

## ⚡ Performance & Feel
- **Fluid Typography**: Implemented `clamp()` for consistent sizing across all viewports.
- **Buttery Smooth Scroll**: Optimized Lenis lerp values for a premium, heavy-weight scrolling feel.
- **Synchronized Transitions**: GSAP ScrollTrigger ensures content reveals exactly as the user interacts.
